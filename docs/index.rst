CEN Bot
=======

CEN Bot is a Discord bot built and maintained by `Collegiate Esports Network LLC`_.
It provides guild activity logging, member welcome messages, YouTube upload alerts,
Twitch live alerts, an XP system, radio playback, and various utility commands.

.. _Collegiate Esports Network LLC: https://collegiateesportsnetwork.org
