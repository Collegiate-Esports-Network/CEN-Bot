Moderation
==========

Guild activity logging with five verbosity levels (0–4). Per-guild config is
loaded into an in-memory cache at startup — no DB queries during events.
Provides context menus for reporting messages/users, and ``/moderation``
slash commands for configuration and moderation actions.
